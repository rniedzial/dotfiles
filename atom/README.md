# Summary

Make your life easier and install Atom. Then install the following packages to help you work with CloudFormation, Python and NodeJS code:

## CloudFormation YML Support
* atom-cform-yaml
* language-yaml-cloudformation
* linter-js-cloudformation-yaml

## Python Support
* language-python
* autocomplete-python
* python-tools
* linter-flake9

## General
* sync-settings
* git-plus

## Chef
* language-ruby
* language-chef
* language-habitat
* language-rspec
* language-inspec
* linter-rubocop
* linter-foodcritic
